### User Service

user authentication, login, logout

##### <u>Endpoints</u> 

- /api/user/all - GET 
  - Returns all users
- /api/user/create - POST
  - Creates a user
- /api/user/login  - POST
  - Allows user to log in
- /api/user/logout  - POST
  - Allows user to logout
- /api/user/`<username>`/exists  - GET
  - Checks if a user exists in the database
- /api/user/  - GET
